Calculator

A simple calculator utilizing Javascript's computing to evaluate the input and handle keyboard events to affect the DOM. Stylized with CSS Grid and Flexbox. An embedded iframe is currently displayed so you can interact with it, provided you're not on a narrow viewport on mobile.

Still need to implement a proper algorithm for input parsing input (Shunting-yard) for more calculator-like feedback.

Etch-a-Sketch

JS-based etch-a-sketch that dynamically creates divs to act as pixels. Another embedded iframe is in the picture on desktop-view to engage with.

Old Apple

My rendition of the old version of the apple website from years ago, doing a lot of layering of box-shadows to get the navigation bar to display correctly and some funky logic to get it to perform as it should.

Looks pretty good as long as you're on a webkit browser (Opera, Safari, Chrome, Brave) since the rendering of shadows are so different with the Mozilla Firefox rendering implementation.
